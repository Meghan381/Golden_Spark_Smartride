# Golden-Spark SmartRide

A web-based ride-sharing platform for booking rides, managing drivers, and admin operations.

## Features

### For Passengers
- Book rides with pickup/dropoff locations
- Choose vehicle type (Economy, Premium, Luxury)
- Real-time fare estimation
- View ride history and ratings
- Loyalty points system

### For Drivers
- Accept/decline ride requests
- Track earnings and ride statistics
- Toggle online/offline status
- Driver dashboard with real-time updates

### For Admins
- Manage users, drivers, and passengers
- View revenue reports and analytics
- Security logs and activity monitoring
- Platform settings and fare configuration

## Tech Stack

- HTML5
- CSS3
- JavaScript (LocalStorage for data persistence)
